# Writing Craft — Писательский справочник

Публичный аудит материалов по писательскому мастерству.

## Что здесь

- **WRITING-CRAFT.md** — единый справочник (~15KB) для загрузки перед писательскими задачами
- **KB/** — тематическая база знаний, структурированная по проблемам, а не по книгам

## Источники

Выжимки из:
- Роберт Макки — «История на миллион», «Экшен», «Персонаж», «Диалог»
- Стивен Кинг — «Как писать книги»
- Уильям Зинсер — «Как писать хорошо» *(нон-фикшн, голос, стиль, переписывание)*
- Сториномика (Макки/Джерас)

## Структура KB

```
KB/
├── elements/     — элементы истории (сцена, персонаж, конфликт, тема...)
├── problems/     — частые проблемы (диалог, темп, show/tell, экспозиция...)
└── genres/       — жанровые особенности (экшен, драма)
```

## Принцип

Не «кто что сказал», а «как решить X». Проблемно-ориентированная структура.
