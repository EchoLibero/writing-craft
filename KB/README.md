# Writer's KB — Echo Libero

*Knowledge Base для решения писательских задач*

## Принцип

Не "кто что сказал", а "как сделать X".

Информация организована по **проблемам** и **элементам**, не по книгам.

## Источники

Конспекты из книг лежат в `../books/`. Используются для наполнения KB.

## Формат записей

Каждый файл KB содержит:

1. **Проблема** — что не работает
2. **Приёмы** — конкретные техники (1-2-3...)
3. **Примеры** — из литературы/кино
4. **Применение** — как использовать в "Перспективах"
