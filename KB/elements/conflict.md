# Конфликт

*Двигатель истории*

## Ключевые принципы

1. **Без конфликта нет истории.** Если всё хорошо — читать неинтересно.
2. **Конфликт — это столкновение ценностей.** Положительное vs отрицательное: жизнь/смерть, любовь/ненависть, свобода/рабство.
3. **Эмоция = побочный эффект перемен.** Пока ценности в балансе — читатель спокоен. Когда ценность меняет заряд — читатель чувствует.
4. **Конфликт должен расти.** Каждый следующий этап — выше ставки, сильнее опасность.

## Приёмы

### 1. Ценность в каждой сцене
Каждая сцена — изменение заряда ценности:
- Начало: X положительно (или отрицательно)
- Конец: X меняет заряд

Если заряд не изменился — сцены не было.

### 2. Четыре уровня конфликта
| Уровень | Пример | Конфликт |
|---------|--------|----------|
| Физический | Жизнь/смерть | Борьба за выживание |
| Социальный | Справедливость/несправедливость | Борьба за статус, власть |
| Личный | Любовь/ненависть | Межличностные отношения |
| Внутренний | Истина/ложь | Борьба с собой |

### 3. Триада конфликта
**Герой ↔ Злодей ↔ Жертва**

- Герой → спасает
- Злодей → угрожает
- Жертва → не может спасти себя

### 4. Прогрессия конфликта
Каждый этап:
- Выше ставки
- Сложнее решения
- Ближе к финалу

### 5. Поворотные моменты
Момент, когда ожидания героя разбиваются:
- Он действовал → ожидал X
- Получил Y (неожиданное)
- Новое направление

### 6. Кризис и кульминация
**Кризис** — момент максимального давления. Герой делает выбор.
**Кульминация** — результат выбора.

## Анти-паттерны

- ❌ **Конфликт на пустом месте** → Персонажи спорят о ничём → Ставки
- ❌ **Статичный конфликт** → Ничего не меняется → Рост
- ❌ **Конфликт без последствий** → Победил → Победил → Скучно → Поражение
- ❌ **Слишком много конфликтов** → Второстепенный → Один главный
- ❌ **Злодей не злодей** → Не с кем конфликтовать → Могущественный антагонист

## Примеры из произведений

- **«Крепкий орешек»** — один этаж, один герой vs many антагонисты
- **«Властелин колец»** — глобальный (мир) + личный (Фродо vs Кольцо)
- **«Бегущий по лезвию»** — внутренний (человек ли я?) + физический (выживание)

## Применение к роману "Перспективы"

- [ ] Главный конфликт: кто vs кто?
- [ ] Какой уровень: физический, социальный, личный, внутренний?
- [ ] Как ставки растут от главы к главе?
- [ ] Есть ли точка максимального кризиса?
- [ ] Какая ценность в основе?
