# Побуждающее происшествие

*Момент, когда всё меняется*

## Ключевые принципы

1. **Это точка невозврата.** После побуждающего происшествия герой не может вернуться к прежней жизни.
2. **Идеально — как можно скорее, но не раньше максимального воздействия.**
3. **Включает: злодея, замысел, макгаффин.** Герой обнаруживает → начинает действовать.
4. **Меняет заряд ценности.** Если до — жизнь героя в балансе. После — дисбаланс.

## Приёмы

### 1. Структура побуждающего происшествия
```
Герой живёт в балансе
↓
Обнаруживает злодея / замысел / макгаффин
↓
Жизнь переворачивается
↓
Герой приступает к действию
```

### 2. Варианты по сложности
**Без экспозиции** → побуждающее сразу
**Средний контекст** → краткая экспозиция → происшествие
**Сложный мир** → экспозиция → происшествие в первом акте

### 3. Знакомство с злодеем
Не все знают злодея заранее:
- Злодей появляется первым → герой узнаёт
- Герой обнаруживает след → не знает злодея
- Фальшивый злодей → настоящий позже

### 4. Макгаффин
Объект, нужный злодею. Может быть:
- Статичный (информация)
- Динамичный (жертва)

### 5. Размер события
Пропорционально миру истории:
- Маленький мир → личная угроза
- Большой мир → город/мир/вселенная

### 6. Герой — не жертва
Герой ДЕЙСТВУЕТ в ответ. Он не просто «получает» — он реагирует.

## Анти-паттерны

- ❌ **Слишком поздно** → читатель скучает до
- ❌ **Без контекста** → непонятно, почему важно
- ❌ **Герой пассивен** → событие происходит с ним, не им
- ❌ **Необратимо без драмы** → формальность

## Примеры из произведений

- **«Крепкий орешек»** — Грубер захватывает здание. Всё, точка.
- **«Звёздные войны»** — Люк узнаёт о Лее и Вейдере
- **«Терминатор»** — «Come with me if you want to live»
- **«Касабланка»** — Ильза входит в бар
