# Сеттинг (место действия)

*Мир как активная сила истории*

## Ключевые принципы

1. **Сеттинг — не декорация.** Место действия влияет на конфликт, ограничивает персонажей, создаёт возможности.
2. **Сеттинг раскрывает персонажей.** Где живёт человек — отражение того, кто он.
3. **Выбрать конкретное, отбросить остальное.** Один квартал важнее, чем весь город.
4. **Мир = характер.** Город враждебен? Персонажи агрессивны. Город холоден? Персонажи эмоционально закрыты.

## Приёмы

### 1. Сеттинг как ограничение
Место создаёт правила:
- Лабиринт → ограничение движения
- Город → правила улиц
- Океан → технологии vs стихия

### 2. Сеттинг как возможность
Место создаёт ситуации:
- Канализация «Крепкого орешка» → уникальные действия
- Бурдж-Халифа → уникальные возможности

### 3. Характерные детали
Не всё описание — только то, что характерно:
- Не весь бар — только стойка, запах, свет
- Не весь город — только запах гари, цвет неба

### 4. Кинетический сеттинг
Сеттинг, который сам создаёт действие:
- Ледяная буря
- Крутой склон
- Нестабильное здание

### 5. Контраст
Сеттинг, который противоречит действию:
- Спокойное место → насилие
- Опасное место → нежность

### 6. Weltanschauung персонажа
Мир через призму восприятия:
- Ребёнок видит одно
- Взрослый — другое
- Злодей — третье

## Анти-паттерны

- ❌ **Описание вместо действия** → «Был красивый город с...» → Мир через персонажа
- ❌ **Избыточные детали** → Потеря в лесу → 3-4 характерных детали
- ❌ **Сеттинг как scenery** → Красиво, но не работает → Сеттинг → конфликт
- ❌ **Всевидящий narrator** → Все знает о мире → Персонаж открывает мир
- ❌ **Статичный сеттинг** → Ничего не меняется → Сеттинг evolves with story

## Примеры из произведений

- **«Крепкий орешек»** — недостроенный небоскрёб как character: возможности + ограничения
- **«Не угаснет надежда»** — океан как антагонист: безжалостный, равнодушный
- **«Бегущий по лезвию»** — город ночной, дождливый, неоновый: мир будущего, который не наступит
