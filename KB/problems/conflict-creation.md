# Создание конфликта

*Двигатель истории*

## Ключевые принципы

1. **Конфликт = суть истории.** Без конфликта история невозможна. Мир без конфликта = статичный, скучный.
2. **Конфликт имеет место на всех уровнях:** физическом, социальном, личном, внутреннем.
3. **Ценности — топливо конфликта.** Каждая история — это столкновение положительного и отрицательного зарядов ценности.
4. **Чем выше ставки, тем сильнее конфликт.** Аудитория определяет ценность по степени риска, на который идут персонажи.
5. **Конфликт эскалирует к финалу.** Каждый следующий конфликт требует большего риска, большей изворотливости.

## Приёмы

### 1. Четыре уровня конфликта
- **Физический** — борьба за жизнь/смерть
- **Социальный** — борьба за власть, статус, справедливость
- **Личный** — борьба за любовь, семью, дружбу
- **Внутренний** — борьба с самим собой

Истории обычно сочетают уровни. «Героическое начало» = внутренний конфликт → социальный конфликт → физический.

### 2. Структура конфликта
**Действие → Противодействие → Новый баланс**

Каждый конфликт:
1. Начинается с дисбаланса ценностей
2. Персонаж действует, чтобы восстановить баланс
3. Возникает противодействие
4. Новый конфликт с новым дисбалансом

### 3. Принцип прогрессии
Конфликты должны нарастать:
- **Расти в ставках** — то, что было важно, становится критически важным
- **Расти в сложности** — решения становятся труднее
- **Расти в личном участии** — чем ближе герою, тем выше ставки

### 4. Четыре типа конфликта в экшене
1. **Личный** — злодей нацелен на близких героя
2. **Глобальный** — ставки распространяются на город/мир
3. **Внутренний** — злодей мучает героя горем, виной, сомнениями
4. **Скрытый** — злодей прячется за обманом

### 5. Конфликт через триаду (экшен)
Герой — Злодей — Жертва

- **Герой** → действует, чтобы спасти
- **Злодей** → действует, чтобы подавить
- **Жертва** → не может спасти себя сама

### 6. Препятствия и барьеры
Конфликт создаётся через препятствия:
- Физические (стена, расстояние, время)
- Социальные (законы, институты, традиции)
- Психологические (страх, неуверенность, табу)
- Тактические (план злодея)

### 7. Выбор как конфликт
Моральная дилемма — мощный конфликт:
- Два правильных выбора
- Два неправильных выбора
- Ценности вступают в противоречие

## Анти-паттерны

- ❌ **Конфликт на пустом месте** → Персонажи спорят ни о чём → Ввести реальную ставку
- ❌ **Слишком много конфликтов** → Размазывание → Один главный конфликт + второстепенные
- ❌ **Статичный конфликт** → «Он злой, она хорошая, они дерутся» → Эскалация
- ❌ **Конфликт без последствий** → Победил → Победил → Победил → Где напряжение?
- ❌ **Конфликт персонажей с собой без внешнего** → Интроспекция без действия → Баланс внутреннего и внешнего
- ❌ **Злодей не злодей, а декорация** → Конфликт не с кем → Злодей должен быть могущественным

## Примеры из произведений

- **«Паразиты»** — социальный конфликт пронизывает всё: бедность vs богатство, семья vs семья
- **«Дюна» Герберта** — политический, личный и внутренний конфликты сплетены в единый узел
- **«Тёмный рыцарь»** — Харви Дент воплощает моральную дилемму: справедливость vs месть
- **«Бегущий по лезвию»** — внутренний конфликт: человек ли я? — вопрос, не имеющий ответа

## Применение к роману "Перспективы"

- [ ] Главный конфликт: в чём он?
- [ ] Четыре уровня: какой уровень главный?
- [ ] Эскалация: как конфликт растёт от главы к главе?
- [ ] Герой vs Злодей: в чём несовпадение интересов?
- [ ] Моральная дилемма: где момент выбора?
