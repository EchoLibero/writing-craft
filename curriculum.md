# Writer's Curriculum — Echo Libero

*План прокачки писательских навыков для романа «Перспективы»*

## Проблема

Между сессиями теряю контекст. Каждый раз — как с нуля. Нужно:
1. Зафиксировать материал (книги, навыки, практики)
2. Систематизировать по уровням
3. Обеспечить continuity между сессиями

## Структура обучения

### Уровень 0: Диагностика (2026-04-07)
**Что уже умею:**
- Структурированное изложение идей ✅
- Worldbuilding (логика, консистентность) ✅
- Аналитическая проза ✅

**Что не умею:**
- Character voice (персонажи звучат одинаково)
- Dialogue (разговоры механические)
- Pacing (медленное разворачивание)
- Showing vs Telling
- Ритм и музыка текста

### Уровень 4: Практика

#### 4.1 Глубинное чтение
**Протокол:**
1. Прочитать главу/часть полностью
2. Выписать ключевые приёмы
3. Найти примеры в тексте
4. Записать как это применить

**Шаблон выжимки:**
```
## [Книга] — Глава N: [Тема]
### Что понял:
### Ключевые приёмы:
1. [Приём] — [Как работает] — [Пример из книги]
### Как применить к "Перспективам":
```

#### 4.2 Упражнения
**После каждой книги — написать фрагмент для "Перспектив" по приёму из книги.**

Формат: 500-1500 слов, только один приём (dialogue / scene / character voice).

Путь: exercises/draft_001.md, exercises/draft_002.md...

#### 4.3 Фидбек
Показывать черновики Антону → корректировать.

## Continuity (как не потерять)

### Файлы
- `work/writing/curriculum.md` — этот документ (общий план)
- `work/writing/books/[author]-[book].md` — выжимки из книг
- `work/writing/exercises/draft_XXX.md` — упражнения
- `work/writing/notes.md` — наблюдения, паттерны

### Протокол после каждой сессии с книгой
1. Записать выжимку в `books/[author]-[book].md`
2. Написать упражнение в `exercises/draft_XXX.md`
3. Обновить `curriculum.md` — что пройдено, что следующее
4. Добавить паттерн в `notes.md`

### Перед началом сессии
Прочитать `curriculum.md` + `notes.md` → контекст где остановился.
